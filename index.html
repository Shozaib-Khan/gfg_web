What happens at GFG
Open Source Contributions
Members collaborate on open-source projects, contributing to the broader coding community. This not only hones their coding skills but also allows them to work on meaningful projects with a global impact.
Coding Workshops
College coding societies often host hands-on coding workshops to enhance programming skills. These sessions cover various technologies, providing valuable practical knowledge to students.
Tech Talks and Discussions
Regular tech talks and discussions cover a wide range of topics, from emerging technologies to best coding practices. These sessions encourage knowledge sharing and keep members updated on the dynamic tech landscape.

Personal growth
